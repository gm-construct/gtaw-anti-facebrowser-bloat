
# GTAW Anti Facebrowser Bloat

A *very* simple extension that allows users to toggle visibility for specific post types on GTA World's Facebrowser platform. Tested on **Chrome** and **Edge**; should work on any Chromium-based browser.

**Post types:**
- Sponsored posts
- Boosted posts
- Stories

Furthermore, the extension allows users to **hide cards** on the right side:
- Trending tags
- Hottest posts
- People & Page suggestions

![Popup Example](https://i.imgur.com/w1MXOsl.png)

## Installation

### Click [here](https://github.com/gm-construct/gtaw-anti-facebrowser-bloat/archive/refs/tags/public-release.zip) to download latest version.

- **1)** Download latest version.
- **2)** Extract the `gtaw-anti-facebrowser-bloat-public-release` folder anywhere.
- **3)** Go to your browser's **Extensions** page.
- **3.1)** On **Chrome**: `chrome://extensions` in the URL bar.
- **3.2)** On **Edge**: `edge://extensions` in the URL bar.
- **4)** Enable **Developer mode** in order to load custom extensions.
*Usually located in the top right.*
![Developer mode toggle](https://i.imgur.com/KmjbslA.png)
- **5)** Click **Load unpacked**, and select `gtaw-anti-facebrowser-bloat-public-release` folder you extracted in *step 2*.
![Load unpacked button](https://i.imgur.com/UgymOIa.png)

**Done!**

To use it, go to Facebrowser, open the **Extensions** popup on your browser, and select the extension.
![Extension in the list](https://i.imgur.com/6ZyJdEX.png)

A popup should open where you can toggle which post types to hide.
